SQL AI Copilot - Windows 简化版可执行文件包
========================================

版本信息：
- 编译时间：周一 2025/07/28 14:37:15.38
- 目标系统：Windows 10/11
- 编译器：MinGW-w64
- 版本：简化版（仅依赖Windows系统库）

文件说明：
==========
- sql_ai_copilot.exe：主程序（可执行文件）
- 启动程序.bat：推荐启动方式（包含错误检查）
- config\ai_models.json：AI模型配置文件
- include\：头文件（开发参考用）
- docs\：文档文件夹

使用方法：
==========
1. 双击"启动程序.bat"（推荐）
2. 或直接运行 sql_ai_copilot.exe

首次使用：
==========
1. 确保网络连接正常
2. 检查 config\ai_models.json 中的API密钥
3. 运行程序并按照提示操作

功能说明：
==========
- SQL语句输入（支持多行）
- 执行计划分析
- AI智能优化建议
- 多轮交互问答

系统要求：
==========
- Windows 10/11
- 网络连接
- 至少 50MB 可用内存
- 支持 HTTPS 连接

简化版特性：
==========
- 仅依赖Windows系统库（winhttp.lib）
- 无需安装额外的开发库
- 更小的文件体积
- 更好的兼容性

故障排除：
==========
1. 网络问题：
   - 检查防火墙设置
   - 确认网络连接正常

2. API问题：
   - 检查 config\ai_models.json 中的API密钥
   - 确认API密钥有效且有足够配额

3. 程序崩溃：
   - 检查配置文件格式
   - 确认系统兼容性
   - 查看错误代码

技术支持：
==========
如遇到问题，请提供以下信息：
- 错误信息或错误代码
- 操作系统版本
- 网络连接状态
- 配置文件内容（隐藏API密钥）

更新说明：
==========
如需更新程序，请：
1. 备份当前配置文件
2. 下载新版本
3. 替换可执行文件
4. 恢复配置文件
